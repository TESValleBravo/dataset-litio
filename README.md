# dataset-litio
Este repositorio contiene un dataset procesado para experimentos de selección de atributos y clasificación mediante algoritmos híbridos clásico-cuánticos. El conjunto de datos fue construido a partir del dataset original **eVTOL Battery Dataset**, disponible públicamente en KiltHub/Figshare mediante el DOI:  https://doi.org/10.1184/R1/14226830
